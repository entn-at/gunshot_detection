# Gunshot Detection

#### python 2.7

### Install requisites

```python
pip install -r requirements.txt

```

### Initial setup

```python
python setup.py

```

### Features
- Chroma Stft (Default - 12 coeff)
- Melspectrogram (128 coeff)
- MFCC (Default - 20 coeff)
- RMSE
- Spectral Bandwidth
- Spectral Centroid
- Spectral Contrast
- Spectral Rolloff
- Poly features (Default - 1 order)
- ZCR

### Vector Reduction
- Mean
- IQR
- Variance

### Feature Reduction
- PCA (Default - 10 features)

### Neural Network
- Multilayer Perceptron
